🌊 BFS — 너비 우선
from collections import deque
def bfs(graph, start):
    visited = set([start])
    q = deque([start])
    while q:
        u = q.popleft()
        for v in graph[u]:
            if v not in visited:
                visited.add(v)
                q.append(v)
🌿 DFS — 깊이 우선
def dfs(graph, u, visited=None):
    if visited is None: visited=set()
    visited.add(u)
    for v in graph[u]:
        if v not in visited:
            dfs(graph, v, visited)
    return visited

def find(x):
    if parent[x]!=x:
        parent[x]=find(parent[x])
    return parent[x]
def union(a,b):
    a,b=find(a),find(b)

프로세스 vs 스레드
프로세스: 독립 메모리(코드/데이터/힙/스택), Context Switch 비용 큼, IPC 필요
스레드: 코드/데이터/힙 공유, 스택 독립, Context Switch 빠름, 동기화 필요
CPU 스케줄링
비선점: FCFS, SJF
선점: Round Robin(Time Quantum), SRTF, Priority
기아(Starvation) 해결: Aging
교착상태 (Deadlock)
4가지 조건: 상호배제 · 점유대기 · 비선점 · 순환대기
해결: 예방 / 회피(은행원 알고리즘) / 탐지 및 회복
메모리 관리
페이징: 고정 크기, 외부 단편화 없음
